# Users CRUD NT

A full-stack users CRUD application.

## Tech Stack

### Frontend

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Radix UI**
- **shadcn/ui**
- **react-hot-toast**
- **lucide-react**
- **uuid**
- **tailwind-merge**
- **axios**
- **eslint**

### Backend (API)

- **Node.js**
- **Next.js Server**
- **MongoDB**
- **Mongoose**
- **eslint**

## Features

- Create, read, update, and delete users
- Modern UI with shadcn/ui and Radix UI
- Toast notifications
- RESTful API with MongoDB

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure environment variables.
4. Run the development server:
   ```bash
   npm run dev
   ```
